# 🏎️ ESOGÜ Racing - Dashboard Design

Bu depo, Formula Student aracımızın pilot arayüz (UI) tasarımlarını ve Nextion Editor projelerini barındırır.

### 🎯 Hedefler
- **Okunabilirlik:** Yüksek hız ve titreşim altında pilotun veriyi anlık görmesi.
- **Kritik Veriler:** Vites, RPM, Su Sıcaklığı ve Batarya Voltajı takibi.
- **Görsel Uyarılar:** Yağ basıncı veya aşırı ısınma durumunda tam ekran uyarı sistemleri.

### 🛠️ Araçlar
- Nextion Editor v1.6x
- Adobe Illustrator / Photoshop (İkon Tasarımı)

> **Not:** Tasarımlar Nextion Intelligent serisi ekranlar için optimize edilmiştir.
